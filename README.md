## Run test cases
    $ npm install karma --save-dev
	$ npm install karma-jasmine karma-chrome-launcher --save-dev
	$ npm install -g karma-cli

	Start server:
	$ karma start
	Run tests:
	$ karma run

The you can access your browser with url: http://localhost:8080
